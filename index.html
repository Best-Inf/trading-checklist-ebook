Chapter 01

วันนี้คุณควรเทรดจริงไหม?

ก่อนถามว่า Buy หรือ Sell ทุกเช้าที่เราวางแผน ให้ถามก่อนว่า:

  • วันนี้ใจนิ่งพอไหม
  • นอนพอไหม
  • มีเวลาคิดตามแผนไหม
  • เพิ่งแพ้แล้วยังมีอารมณ์เอาคืนไหม
  • กำลังรีบพิสูจน์ตัวเองอยู่ไหม
Key Insight

ถ้าคำตอบคือ "ยังไม่นิ่ง" — ไม้ที่ดีที่สุดของวันนี้ อาจเป็น ไม่เข้า

การไม่เทรด ไม่ใช่พลาดโอกาสเสมอไป
บางวัน มันคือการรักษาทุนและสติไว้ก่อน

Chapter 02

ตอนนี้ตลาดกำลังเล่นเกมแบบไหน?

ก่อนเข้าไม้ คุณต้องตอบให้ได้ก่อนว่า:

ตลาดกำลังขึ้น · ตลาดกำลังลง · หรือตลาดกำลังแกว่งในกรอบ

Warning

ถ้าตอบไม่ได้ แปลว่าคุณยังไม่รู้ว่ากำลังเล่นเกมอะไร

  • ตลาดขึ้น → อย่ารีบสวนเพราะคิดว่า "ขึ้นมาเยอะแล้ว"
  • ตลาดลง → อย่ารีบช้อนเพราะคิดว่า "ราคาถูกแล้ว"
  • ตลาดแกว่ง → อย่าไล่ราคากลางกรอบเพราะกลัวตกรถ

จุดเข้าอาจทำให้คุณได้เปิดออเดอร์
แต่สภาวะตลาด เป็นตัวบอกว่าไม้นั้นมีความได้เปรียบไหม

Chapter 03

ไม้นี้มาจากแผน หรือมาจากอารมณ์?

ก่อนเข้าไม้ เขียนให้ได้ 1 ประโยค:

Template

"ผมจะเข้าไม้นี้ เพราะ ________"

ถ้าคำตอบคือ:

  • กลัวตกรถ
  • อยากเอาคืน
  • เห็นคนอื่นเข้า
  • กราฟดูเหมือนจะไป
  • รู้สึกว่ารอบนี้ต้องได้
Stop

ให้หยุดก่อน — อารมณ์มันไว้มาก แต่เราต้องตั้งสติให้ดี
เพราะนั่นไม่ใช่แผน นั่นคืออารมณ์ ที่ปลอมตัวเป็นโอกาส

ไม้ที่ดี ควรอธิบายได้แบบเรียบง่าย
ไม่ใช่ต้องหาเหตุผลมาเข้าข้างตัวเอง หลังกดไปแล้ว

Chapter 04

จุดเข้ามีเหตุผลพอไหม?

คุณไม่จำเป็นต้องมีเครื่องมือเยอะ แต่ก่อนเข้า อย่างน้อยควรตอบได้ว่า:

  • เข้าใกล้จุดสำคัญไหม
  • มีแรงตอบสนองจากราคาหรือยัง
  • กำลังเข้าเร็วไปไหม
  • กำลังไล่ราคาอยู่ไหม
  • ถ้ารออีกนิด แผนจะชัดขึ้นไหม

หลายคนแพ้ ไม่ใช่เพราะไม่มีเครื่องมือ
แต่เพราะเข้าไม้ ตอนที่ "ยังไม่ควรเข้า"

Remember

ถ้าราคาอยู่กลางทาง ไม่มีจุดอ้างอิง ไม่มีเหตุผลชัด อย่าเพิ่งเรียกมันว่า Setup
บางครั้ง การรอให้ชัดขึ้นอีกนิด คือต้นทุนที่ถูกกว่าการโดน SL

Chapter 05

SL วางจากโครงสร้าง หรือวางจากความกลัว?

ก่อนเข้าไม้ ต้องรู้ก่อนว่า:

  • ถ้าผิดทาง จะออกตรงไหน
  • จุดไหนบอกว่าแผนนี้ใช้ไม่ได้แล้ว
  • ถ้าถึงจุดนั้น จะยอมออกจริงไหม
Key Insight

SL ที่ดี ไม่ใช่จุดที่คุณ "ทนขาดทุนไหว"
แต่คือจุดที่บอกว่า มุมมองนี้ผิดแล้ว

ถ้า SL อยู่ ใกล้เกินไป เพราะกลัวเสียเงิน → คุณอาจโดนสะบัดออก ทั้งที่แผนยังไม่เสีย

ถ้า SL ไกลเกินไป เพราะไม่อยากแพ้ → คุณอาจไม่ได้คุมความเสี่ยง แต่กำลังปล่อยให้อารมณ์คุมแทน

Chapter 06

TP มีเหตุผล หรือแค่อยากได้กำไร?

ก่อนเข้าไม้ ให้ถามว่า:

  • ราคามีเหตุผลจะไปถึง TP ไหม
  • มีแนวรับแนวต้านขวางอยู่ไหม
  • ตลาดมีแรงพอไปต่อไหม
  • RR คุ้มกับความเสี่ยงไหม
Key Insight

TP ที่ดี ไม่ใช่จุดที่คุณ "อยากได้"
แต่คือจุดที่ตลาด มีเหตุผลจะไปถึง

ถ้า TP ไกลมาก แต่เหตุผลมีน้อย
นั่นอาจไม่ใช่ความมั่นใจ แต่อาจเป็นความหวัง ที่แต่งตัวเหมือนแผน

Chapter 07

Lot นี้ทำให้คุณยังคิดชัดไหม?

ก่อนกดเข้าไม้ ให้ถามตัวเองตรง ๆ:

  • ถ้าโดน SL จะยังใจนิ่งไหม
  • ถ้าแพ้ไม้นี้ จะยังเทรดตามแผนได้ไหม
  • ถ้าไม้ติดลบ จะเริ่มอยากเลื่อน SL ไหม
  • ถ้าแพ้ จะอยากเพิ่ม lot เพื่อเอาคืนไหม
Warning

ถ้า lot ทำให้คุณเริ่มภาวนา แทนที่จะทำตามแผน
แปลว่า lot ใหญ่เกินไป

ไม้ที่ดี ไม่ใช่ไม้ที่กำไรเยอะที่สุด
แต่คือไม้ที่ แม้ผิดทาง คุณยังควบคุมตัวเองได้

เพราะถ้า lot ทำให้ Mindset พัง ต่อให้ Setup สวย ก็ไม่ใช่ไม้ที่ดี

Chapter 08

ถ้าแพ้ไม้นี้ คุณจะเรียนรู้อะไร?

ก่อนเข้าไม้ ให้เตรียมจดไว้เลยว่า:

  • เข้าเพราะอะไร
  • ตอนเข้ารู้สึกอะไร
  • ตลาดอยู่สภาวะไหน
  • SL / TP วางตรงไหน
  • ถ้าแพ้ จะถือว่าเรียนเรื่องอะไร
Core Principle

ขาดทุนแล้วไม่จด = จ่ายซ้ำ
ขาดทุนแล้วเห็นตัวเอง = เริ่มเรียน

ตลาดไม่ได้มีใบเกรดให้คุณ
ดังนั้น Journal คือวิธีที่คุณต้องเฉลยให้ตัวเอง

✅ Quick Checklist ก่อนกดเข้าไม้

  • มี TP ที่มีเหตุผล
  • Lot ไม่ใหญ่จนทำให้ใจเสีย
  • ถ้าเงื่อนไขไม่ครบ ยอมไม่เข้าได้
